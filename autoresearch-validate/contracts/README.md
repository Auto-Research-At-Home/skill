# Contracts bundle (`autoresearch-validate`)

This directory mirrors [`../../autoresearch-create/contracts/0g-galileo-testnet/`](../../autoresearch-create/contracts/0g-galileo-testnet/) so **`autoresearch-validate` does not depend on `autoresearch-create` at runtime.** Scripts resolve `deployment.json` and ABI JSON from here by default.

## Sync

When Galileo ABIs or addresses change, copy from **`autoresearch-create/contracts/0g-galileo-testnet/`** into this folder (same policy as [`autoresearch-mine/contracts/README.md`](../../autoresearch-mine/contracts/README.md)).
